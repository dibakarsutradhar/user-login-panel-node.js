# A User/Member Register, Login and Logout Panel
___

# Technology Used - 

* **Node.js**
* **Expressjs**
* **Passportjs**
* **Bcryptjs Hashing Library**
* **Flash**
* **MongoDB for Database**
* **Jade(PUG)**
___

## To Start this project

```
npm install
```

```
npm start
```
